# Bubble Game

An interactive browser-based game where players must quickly find and click the correct number displayed on the screen before time runs out.

The game tests speed, focus, and reaction time while dynamically updating scores and game states using JavaScript.

---

## 🎮 How to Play

1. A **target number** is displayed in the hit section.
2. Multiple numbered bubbles appear on the screen.
3. The player must click the bubble matching the target number.
4. ✅ Correct click:

   * Score increases
   * New target number appears
   * Bubbles reshuffle
5. ❌ Wrong click:

   * Game Over
6. ⏰ When time finishes:

   * Final score is displayed
   * "Play Again" button appears

## 🧠 Game Logic

* Target numbers are generated randomly.
* Bubbles are dynamically created using JavaScript.
* Click events validate whether the selected number matches the target.
* Timer controls the game duration.
* Score updates instantly based on correct selections.
